# Neural Networks

Das Projekt verwendet als Grundlage eine "Neural Network" Implementierung von David Miller.

## Das "Simple C++ Neural Network" Tutorial

Es gibt ein sehr bekanntes, minimalistisches Tutorial von David Miller, das zeigt, wie man ein Netz in C++ baut. 
Fokus: Er nutzt zwar `std::vector`, aber du kannst seine Logik leicht auf deine Matrix-Klasse ummünzen.
