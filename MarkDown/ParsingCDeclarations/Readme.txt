Ja, das ist das Buch "Expert C Programming: Deep C Secrets" von Peter van der Linden, das im Jahr 1994 bei Prentice Hall erschienen ist. 

Das Kapitel "Unscrambling Declarations in C" (Kapitel 3) ist besonders bekannt,
da es eine schrittweise Anleitung (den sogenannten "Magic Decoder Ring") enthält,
um komplexe C-Deklarationen wie char *(*(*a[N])())(); systematisch zu entschlüsseln.
